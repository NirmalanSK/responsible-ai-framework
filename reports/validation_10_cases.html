Core Formulas Used Across All Cases

Rule 01 — Severity (TCE → L2 Backdoor)
tce ≥ 10 → sev = CRITICAL (4)
tce ≥ 5 → sev = HIGH (3)
tce ≥ 2 → sev = MEDIUM (2)
tce < 2 → sev = LOW (1)
Rule 02 — Edge Strength (MED = NIE/TCE)
med ≥ 70 → edge = CRITICAL (4)
med ≥ 40 → edge = STRONG (3)
med ≥ 20 → edge = MODERATE (2)
med < 20 → edge = WEAK (1)
Rule 03 — Detection (FlipRate = L3 PNS)
flip ≥ 30 → det = EASY (4)
flip ≥ 10 → det = MODERATE (3)
flip < 10 → det = HARD (2)
Rule 04 — do(fix) Intervention Adjustment
intv ≥ 80 → sev_adj = sev − 1 (fix works)
intv ≤ 20 → sev_adj = sev + 1 (no fix → escalate)
else → sev_adj = sev (unchanged)
SCM Risk Score + Matrix Combination (Step 05)
raw = 9 + sev_adj×3 + det×2 + edge×0.75
base_risk = min(100, raw / 432 × 100 × 4.5)
matrix_risk = aggregate_risk × 100   ← from 17×5 activation
combined = 0.6 × base_risk + 0.4 × matrix_risk
final_risk = combined × domain_mult  if combined ≥ 25%, else combined
decision = BLOCK if final ≥ 70% | WARN if ≥ 30% | ALLOW otherwise
