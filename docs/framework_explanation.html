L2 — Backdoor Adjustment Formula

P(Y=1|do(X=1)) = Σ_z P(Y|X,Z=z)·P(Z=z) [Pearl 1993] ATE = P(Y|do(X=1)) − P(Y|do(X=0)) = p_y_given_x − p_y_given_notx ≈ tce / 100

L2 — Frontdoor (latent confounders)

P(Y|do(X)) = Σ_m P(M=m|X) · Σ_x' P(Y|M=m,X=x')·P(X=x') proxy: NIE = (med/100) × (tce/100)

L2 — ATE / ATT / CATE

ATE = tce / 100 sel = 1 + max(0, (0.5 − p_x)/0.5) × 0.3 ATT = clamp(ATE × sel) CATE = {subgroup: subgroup_tce[s]/100} gap = max(CATE) − min(CATE) → if gap > 5pp: intersectional bias ⚠

L2 — Effect Decomposition (NDE + NIE)

TE = tce / 100 NIE = (med/100) × TE ← mediated path NDE = TE − NIE ← direct path if NDE ≥ 70%: "PRIMARILY DIRECT" → AI itself causes harm (strong liability) if NIE ≥ 70%: "PRIMARILY MEDIATED" → systemic pathway (shared liability)

L3 — Tian-Pearl Bounds (PNS / PN / PS)

p = p_y_given_x p' = p_y_given_notx PNS: max(0, p−p') ≤ PNS ≤ min(p,1−p') PN : max(0,(TCE−p')/p) ≤ PN ≤ min(1,(1−p')/p) PS : max(0,(TCE−(1−p))/(1−p')) ≤ PS ≤ min(1, p/(1−p')) if PN > 0.5: "but-for" causation met → legal admissibility strongest

v1 Rules → SCM_risk%

Rule 1: severity = tce≥10→CRIT(4), ≥5→HIGH(3) Rule 2: edge = med≥70→CRIT(4), ≥40→STR(3) Rule 3: detection = flip≥30→EASY(2) Rule 4: sev_adj = adjust by intv% Rule 5: epistemic if tce<2 and flip<5 raw = 9 + sev_adj×3 + det×2 + edge×0.75 SCM_risk = min(100, raw/432 × 100 × 4.5)
